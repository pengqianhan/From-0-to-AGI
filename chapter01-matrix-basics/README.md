# 第 1 章：矩阵与线性代数

**一句话目标**：这章结束后，你能用矩阵乘法手写一个神经网络的前向传播，并解释为什么神经网络的每一层本质上就是一次矩阵乘法。

---

## 起点代码

先运行这段代码，看看它输出什么：

```bash
uv run python examples/01_matrix_basics.py
```

不需要现在就看懂每一行。运行完之后，带着下面的问题去问 Claude Code。

---

## 引导问题

这些问题没有标准答案的长度——有时候一句话够了，有时候你需要问好几个来回才能真正弄清楚。判断标准只有一个：**你能不能用自己的话解释给一个完全不懂的人听**。

### 关于向量

1. 向量 `[80, 2, 5.0]` 表示一套房子的三个特征（面积、卧室数、距市中心距离）。如果我想加一套新房子的数据，我需要改变向量的维度吗？
2. 两个向量的点积是一个数。这个数在几何上代表什么？试着让 Claude Code 用图解释给你看。
3. 如果两个向量完全垂直，它们的点积是多少？为什么注意力机制要用点积来衡量"相关性"？

### 关于矩阵乘法

4. 运行 `examples/02_matrix_multiply.py`，里面有三重循环实现矩阵乘法。试着在 Claude Code 里问：为什么不能直接用两重循环？第三层循环在做什么？
5. 矩阵 A 是 `(2, 3)`，矩阵 B 是 `(3, 5)`，相乘结果是什么形状？如果换成 `(3, 2)` 和 `(3, 5)` 会怎样？让 Claude Code 解释这个"维度匹配规则"背后的原因。
6. `A @ B` 和 `A * B` 在 NumPy 里是完全不同的操作。让 Claude Code 举一个例子，说明混淆这两个会导致什么样的 bug。

### 关于神经网络层

7. 运行 `examples/03_linear_layer.py`。找到这一行：`H = X @ W1 + b1`。权重矩阵 `W1` 的每一列代表什么？每一行呢？
8. 如果我把权重矩阵 `W1` 全部初始化为 0，网络还能学习吗？为什么？（先自己猜，再去问）
9. 代码里用 `rng.standard_normal()` 来初始化权重，为什么不用全零或者全一？

## 动手任务

完成以下三个任务，每个都要运行代码验证结果。遇到问题直接问 Claude Code，但要确保你理解了它给你的每一行代码。

**任务 1（基础）**：不用 NumPy，只用 Python 列表，写一个函数计算两个向量的点积。用 `[1, 2, 3]` 和 `[4, 5, 6]` 验证，结果应该是 32。

**任务 2（核心）**：给 `examples/02_matrix_multiply.py` 里的 `matmul` 函数加一个功能：在维度不匹配时，打印出清晰的错误信息，告诉用户哪里出了问题（比如"A 的列数是 3，但 B 的行数是 4，无法相乘"）。

**任务 3（挑战）**：修改 `examples/03_linear_layer.py`，把两层网络改成三层（加一个 `2维 → 3维` 的中间层），让整个前向传播跑通，打印出每一步的张量形状。
