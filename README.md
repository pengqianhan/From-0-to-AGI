# From 0 to AGI：从矩阵到大语言模型的完整学习路径

> *You can outsource your thinking, but you cannot outsource your understanding.*

---

## 这不是一本教科书

市面上不缺解释矩阵乘法的文章，也不缺讲 Transformer 的博客。

这个教程假设你身边有 **Claude Code 或 Codex**，它可以随时回答你的任何问题、帮你运行代码、解释报错。既然如此，把知识一股脑写给你读没有意义——那是在用教科书的方式对待一个拥有私人老师的人。

**这里做的事情不同**：每一章给你一个清晰的目标、一段可以运行的起点代码，然后给你一系列**需要你自己去问、去探索、去弄懂**的问题。

Claude Code 可以帮你思考，但理解只能由你自己建立。

---

## 使用方式

**你需要准备**：
- Claude Code 或 Codex（任选其一）
- Python 基础不是必须的，但有最好

**每章的节奏**：

1. 阅读章节的**一句话目标**——明确这章结束后你应该能做什么
2. 运行**起点代码**，看看它做了什么
3. 拿着章节给出的**引导问题**去问 Claude Code，直到你真正理解
4. 完成**动手任务**——不是选择题，是要你写代码并运行
5. 用 `/chXX` Skill 做最后的自我检验

**关于 Skill**：每章配套一个 Claude Code Skill（`/ch01-matrix` 等）。它不是答案手册，而是一组**检验性问题**——如果你能用自己的话回答它们，说明你真的懂了。

## 快速开始

```bash
# 1. 安装 uv 包管理器（已安装可跳过）
curl -LsSf https://astral.sh/uv/install.sh | sh

# 2. 克隆项目并安装依赖
git clone https://github.com/your-username/From-0-to-AGI.git
cd From-0-to-AGI
uv sync

# 3. 打开第 1 章，按里面的指引开始
```

```
矩阵乘法 → 梯度下降 → 神经网络
                              ↓
              CNN (图像) ←→ RNN (序列)
                              ↓
                      注意力机制
                              ↓
                        Transformer
                              ↓
               预训练模型 → LLM → 对齐
                              ↓
              多模态 ←→ 世界模型 → AGI
```
